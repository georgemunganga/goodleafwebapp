# UI/UX Design Patterns from Reference Designs

## Key Design Elements Observed

### 1. Color Scheme
- **Primary Green Header**: Gradient green header (#2e7146 to #1a5c35)
- **White Content Area**: Clean white background for forms and content
- **Green Accent Buttons**: Solid green (#3CB371) for primary CTAs
- **Light Gray Backgrounds**: Subtle gray (#f5f5f5) for card backgrounds

### 2. Typography
- **Avenir Font**: Clean, modern sans-serif
- **Bold Headlines**: Large, bold titles for section headers
- **Regular Body Text**: Clean readable body text
- **Small Labels**: Uppercase or smaller text for field labels

### 3. Layout Patterns
- **Green Header Bar**: Consistent header with back arrow, title, and menu dots
- **Rounded White Cards**: Content in rounded white containers (radius ~16-20px)
- **Full-Width Buttons**: Green buttons at bottom, full width with rounded corners
- **Vertical Timeline**: For installment schedules with connected dots/lines
- **Progress Bars**: Horizontal segmented progress indicators

### 4. Form Design
- **Bordered Input Fields**: Simple bordered inputs (not filled backgrounds)
- **Floating Labels**: Labels above inputs, not inside
- **Dashed Upload Areas**: Dashed border boxes for file uploads
- **Checkbox with Terms**: Standard checkbox with terms link

### 5. Navigation
- **Bottom Tab Bar**: Home, Score, History, Profile tabs
- **Back Arrow**: Simple left arrow for navigation
- **Three-dot Menu**: Options menu on right side of header

### 6. Cards & Lists
- **Loan Cards**: Show amount, status, date, with "Check Details" link
- **Profile Card**: Avatar, name, email, phone in header area
- **Menu Items**: Icon + text + chevron right pattern

### 7. Status Indicators
- **Progress Circle**: For scores/completion
- **Timeline Dots**: Filled for completed, empty for pending
- **Status Badges**: Colored badges for loan status

### 8. Spacing
- **Generous Padding**: 16-24px padding in cards
- **Consistent Margins**: 16px horizontal margins
- **Section Spacing**: 24px between major sections

## Components to Create

1. **AppHeader** - Green header with back, title, menu
2. **FormInput** - Bordered input with label
3. **PrimaryButton** - Full-width green rounded button
4. **LoanCard** - Loan summary card with details
5. **UploadBox** - Dashed border upload area
6. **TimelineItem** - For payment schedules
7. **MenuListItem** - Icon + text + chevron
8. **ProgressSteps** - Horizontal step indicator
9. **ProfileHeader** - User info card with avatar
10. **BottomNav** - Tab navigation bar
